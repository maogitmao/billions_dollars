# 股票信息爬虫功能说明

## 功能概述

增强版爬虫提供全面的股票信息获取，包括：

### 1. 公司基本信息 🏢
- 公司名称
- 所属行业
- 主营业务（100字摘要）
- 上市日期
- 换手率

### 2. 财务数据 💰
**基础指标：**
- 总市值（亿元）
- 流通市值（亿元）
- 市盈率（PE）
- 市净率（PB）
- 净资产收益率（ROE）

**增强指标：**
- 市销率（PS）
- 市现率（PCF）
- 每股收益（EPS）
- 每股净资产（BVPS）

### 3. 资金流向 💸
**今日资金流向：**
- 主力净流入（万元）
- 超大单净流入
- 大单净流入
- 中单净流入
- 小单净流入

### 4. 股东信息 👥
- 股东户数
- 人均持股
- 前十大股东持股比例

### 5. 研究报告 📊
- 研报标题
- 研究机构
- 研究员
- 评级
- 发布日期
- 摘要（前100字）

### 6. 公告信息 📢
- 公告标题
- 发布日期
- 公告类型
- 公告摘要
- 最多显示20条，展示前10条

### 7. 新闻资讯 📰
- 新闻标题
- 发布日期
- 新闻来源
- 新闻摘要（200字）
- 最多获取20条，展示前10条

## 使用方式

### 在程序中使用

1. 点击"📥 获取信息"按钮
2. 自动获取当前选中股票的所有信息
3. 信息显示在日志区域
4. AI自动分析获取的信息

### 数据来源

- 东方财富网（主要数据源）
- 新浪财经（辅助数据）
- 同花顺（研报数据）

## 数据更新频率

- 公司信息：静态数据，变化较少
- 财务数据：每季度更新
- 资金流向：实时更新
- 股东信息：定期更新
- 研报：实时更新
- 公告：实时更新
- 新闻：实时更新

## 注意事项

1. 网络连接：需要稳定的网络连接
2. 请求频率：避免频繁请求，建议间隔5秒以上
3. 数据准确性：数据来自公开渠道，仅供参考
4. 异常处理：网络异常时会显示错误信息

## 技术实现

- 使用requests库进行HTTP请求
- 支持JSONP解析
- 正则表达式提取HTML数据
- 异步线程避免UI卡顿
- 完善的错误处理机制
