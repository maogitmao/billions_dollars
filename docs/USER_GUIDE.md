# 用户使用指南

## 如何使用增强版信息爬虫

### 第一步：选择股票

在左侧"实时行情"区域，点击任意一只股票，使其高亮显示。

### 第二步：获取信息

点击中间功能区的"📥 获取信息"按钮。

### 第三步：等待加载

- 按钮会显示"正在获取..."
- 状态栏显示进度
- 通常需要5-10秒

## 使用技巧

### 技巧1：定期更新

建议每天更新一次信息，获取最新数据。

### 技巧2：交叉验证

结合多个数据源验证信息：
- 公告 + 新闻 → 验证消息真实性
- 研报 + 资金流向 → 验证机构态度
- 财务数据 + 股东信息 → 评估基本面

### 技巧3：关注重点

不同投资风格关注不同信息：

**短线投资者**：
- 资金流向（主力动向）
- 新闻资讯（热点题材）
- 分时图（当日走势）

**中线投资者**：
- 研究报告（机构观点）
- 公告信息（重大事项）
- K线图（趋势判断）

**长线投资者**：
- 财务数据（估值水平）
- 公司信息（行业地位）
- 股东信息（筹码稳定性）

### 技巧4：AI辅助

充分利用AI分析功能：

```
# 综合分析
"帮我综合分析这只股票的投资价值"

# 风险评估
"这只股票有哪些风险？"

# 对比分析
"对比一下600519和000858"

# 时机判断
"现在是买入的好时机吗？"
```

## 常见问题

### Q1：信息获取失败怎么办？

**A：** 可能原因：
1. 网络连接问题 → 检查网络
2. 数据源暂时不可用 → 稍后重试
3. 股票代码错误 → 检查代码

### Q2：部分信息显示"暂无数据"？

**A：** 正常现象，可能原因：
1. 该股票确实没有相关数据
2. 数据源暂时无法访问
3. 新股上市，历史数据不足

### Q3：AI分析不准确？

**A：** AI分析仅供参考：
1. 基于提供的数据进行分析
2. 不构成投资建议
3. 需结合自身判断

### Q4：如何保存信息？

**A：** 两种方式：
1. 在日志区域选中文本，Ctrl+C复制
2. 截图保存（推荐）

### Q5：可以同时查看多只股票吗？

**A：** 目前一次只能查看一只，但可以：
1. 依次获取多只股票信息
2. 在日志区域查看历史记录
3. 使用AI对比分析

## 注意事项

⚠️ **重要提示**：

1. **数据仅供参考**
   - 来自公开渠道
   - 可能存在延迟
   - 不保证100%准确

2. **不构成投资建议**
   - AI分析仅供参考
   - 需自行判断
   - 投资有风险

3. **合理使用**
   - 避免频繁请求
   - 建议间隔5秒以上
   - 尊重数据源

4. **网络安全**
   - 不要在公共网络使用
   - 注意保护个人信息
   - 定期更新软件
