# 参考数据文件夹

本文件夹存放参考数据和历史数据。

## 文件夹结构

```
reference/
├── historical/          # 历史数据
│   ├── kline/          # K线历史数据
│   ├── quotes/         # 行情历史数据
│   └── indicators/     # 指标历史数据
│
├── reference_data/      # 参考数据
│   ├── stock_list/     # 股票列表
│   ├── industry/       # 行业分类
│   └── financial/      # 财务数据
│
└── downloads/           # 下载的原始数据
```

## 使用说明

### 历史数据
- 用于回测和分析
- 按日期组织：`YYYYMMDD.csv`
- 定期清理过期数据

### 参考数据
- 股票基本信息
- 行业分类数据
- 财务报表数据
- 不经常变动的数据

### 下载数据
- 从数据源下载的原始数据
- 处理前的临时存储
- 可定期清理

## 数据格式

### K线数据格式
```csv
day,open,high,low,close,volume
2024-01-01,10.00,10.50,9.80,10.20,1000000
```

### 行情数据格式
```json
{
  "code": "600000",
  "name": "浦发银行",
  "price": 10.20,
  "change": 0.10,
  "change_pct": 0.99
}
```

## 注意事项

- 大文件使用压缩格式（.gz, .zip）
- 定期备份重要数据
- 清理超过1年的历史数据
- 不要提交到git（已在.gitignore中）
