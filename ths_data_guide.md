# 同花顺数据导入指南

## 方法1：从同花顺导出CSV文件

### 步骤：
1. 打开同花顺客户端
2. 在自选股或行情列表中，选中要导出的股票
3. 右键点击 → 选择"导出数据"或"复制"
4. 将数据保存为CSV文件
5. 将CSV文件放到项目的 `ths_data/` 目录下
6. 文件命名为 `quotes.csv` 或 `realtime_quotes.csv`

### CSV文件格式：
```csv
代码,名称,最新价,涨跌额,涨跌幅,成交量,成交额,最高,最低,今开,昨收
600000,浦发银行,8.50,0.10,1.19,12345678,105000000,8.55,8.45,8.48,8.40
600519,贵州茅台,1650.00,15.50,0.95,8765432,14500000000,1665.00,1645.00,1648.00,1634.50
```

## 方法2：使用同花顺的DDE接口（高级）

如果同花顺支持DDE接口，可以实时获取数据。

## 方法3：通过剪贴板导入

1. 在同花顺中选中股票行情
2. 按 Ctrl+C 复制
3. 在我们的程序中点击"从剪贴板导入"按钮
4. 程序会自动解析数据

## 方法4：使用模拟数据（测试用）

如果暂时无法获取真实数据，程序会自动生成模拟数据用于测试界面功能。

## 推荐方案

**最简单的方式：**
1. 在同花顺中添加自选股
2. 定期导出自选股数据为CSV
3. 将CSV文件放到 `ths_data/quotes.csv`
4. 程序会自动读取并显示

**自动化方案（需要开发）：**
- 编写脚本定期从同花顺导出数据
- 或者使用同花顺的API接口（如果有）
